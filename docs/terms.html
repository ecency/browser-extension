Description of Service

Hive Keeper is a free, open-source browser extension that allows you to:

  • Store Hive blockchain private keys locally on your device (encrypted).
  • Sign and broadcast transactions to the Hive blockchain.
  • Interact with Hive-based decentralized applications (dApps).
  • Manage Hive and Hive Engine token balances.
  • Swap tokens using the Hive internal market and Hive Engine decentralized exchange.
